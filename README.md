# MouseInfo 3D

Starter multi-page React + Vite app with a simple 3D viewer using @react-three/fiber.

Features included in this starter:
- Multi-page routing (Home, Overview, Anatomy/Specs, Gallery, Compare, FAQ, Resources)
- Simple procedural 3D "mouse" model (no external GLTF required)
- JSON data file (src/data/mice.json) for content
- Tailwind CSS for styling

How to run locally:

1. Clone the repo
   git clone https://github.com/Aryansinghhh11/Aryansingh.git
2. Install dependencies
   npm install
3. Run dev server
   npm run dev
4. Open http://localhost:5173

Notes:
- Theme: Light modern (you selected option 2)
